# Entity Framework Core Masterclass - Dashboard Project 🚀

Bu repo, Entity Framework Core (EF Core) kullanarak geliştirilen, ileri seviye LINQ sorguları ve modüler bir Dashboard mimarisi içeren eğitim projesini kapsamaktadır.

## 📌 Proje Hakkında
Bu çalışma kapsamında, kurumsal bir web uygulamasının arka yüz (backend) ve ön yüz (UI) entegrasyonu modern pratiklerle gerçekleştirilmiştir. Proje; veri tabanı yönetiminden, karmaşık veri sorgulama tekniklerine kadar geniş bir yelpazeyi kapsamaktadır.

## 🛠 Kullanılan Teknolojiler
- **Framework:** .NET Core / ASP.NET Core MVC
- **ORM:** Entity Framework Core
- **Database:** MSSQL (LocalDB)
- **UI:** View Components, Partial Views, Dashboard Themes
- **Sürüm Kontrol:** Git & GitHub

## 📑 Öne Çıkan Özellikler & İçerik

### 1. Veri Tabanı ve Mimari
- **Code First Yaklaşımı:** Entity'lerin oluşturulması ve `DbContext` konfigürasyonu.
- **Migration Yönetimi:** Veri tabanı şemalarının düzenli olarak güncellenmesi.
- **Configuration:** Fluent API ve Data Annotations ile tablo ilişkilerinin yönetimi.

### 2. İleri Seviye LINQ Sorguları
Proje içerisinde aşağıdaki kompleks metotlar aktif olarak kullanılmıştır:
- `Intersect`, `Cast` & `OfType`
- `First` & `SingleOrDefault`
- `DefaultIfEmpty` & `Reverse`
- `Entry` metodu ile State takibi

### 3. Modüler UI Yapısı
- **View Components:** Navbar, Sidebar ve İstatistik kartlarının (Card Statistics) bağımsız bileşenler olarak geliştirilmesi.
- **Dashboard Entegrasyonu:** Dinamik Todo listesi, mesajlaşma paneli ve istatistik yönetim ekranları.
